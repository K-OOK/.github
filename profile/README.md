## 💡 Introduce

KOOK은 계속해서 세계로 나아가는 K-FOOD를 내 집에 있는 재료로 즐기고 싶은 외국인들에게,<br>
원하는 재료로 만들 수 있는 한식 레시피를 제공하는 서비스입니다.

**🥦 재료 기반 한식 추천**: commerce data 기반 최다 판매 식재료 혹은 직접 입력한 식재료를 기반으로 조리가능한 한식을 레시피를 제공합니다.<br>
**🔥 트렌드 중인 한식 레시피 제공**: 해외 커뮤니티에서 트렌드 중인 한식 리스트를 제공하고, 해당 한식의 레시피를 제공합니다.<br>
